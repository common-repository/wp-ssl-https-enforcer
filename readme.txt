A simple wordpress plugin to enforce HTTPS

== Description ==

A simple wordpress plugin to enforce HTTPS on the domain and it's contents if the SSL certificate is installed for the domain.

= Key Features =

*   Easy to use and implement.

= Demo =

[Click here for demo](https://www.base29.com/)

== Installation ==


1. Upload plugin folder `wp-ssl-enforcer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
